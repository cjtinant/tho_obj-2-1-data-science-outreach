## PRE-SURVEY
### "Before We Begin"
*Shown at top of form:*
> This short survey takes about 2 minutes. Your answers help us improve
> the workshop and report to our funders. All responses are anonymous.

---

**Question 1 — About you** *(Short answer, optional)*
```
What is your role? (e.g., student, educator, community member)
```

**Question 2 — Prior experience** *(Multiple choice, required)*
```
Have you ever written code before?

○ Yes — I code regularly
○ Yes — I have tried it a few times
○ A little — I have seen code but never written it
○ No — this is new to me
```

**Question 3 — What is data science?** *(Paragraph, required)*
```
In your own words, what do you think data science is?
(There is no wrong answer — just write what comes to mind.)
```

**Question 4 — Confidence with data** *(Linear scale 1–5, required)*
```
How confident do you feel working with data right now?

1 = Not confident at all
5 = Very confident
```

**Question 5 — Expectations** *(Multiple choice, required)*
```
What are you most hoping to get out of today's workshop?

○ Learn something new about data or coding
○ See what data science actually looks like
○ Get ideas for my own work or studies
○ Just curious — I am here to explore
○ Other (please describe)
```

## POST-SURVEY
### "Before You Go"
*Shown at top of form:*
> One more short survey — about 3 minutes. Your feedback directly shapes
> future workshops. Thank you for being here today.

---

**Question 1 — Confidence with data** *(Linear scale 1–5, required)*
```
How confident do you feel working with data right now?

1 = Not confident at all
5 = Very confident
```
*Note: This is the same question as Q4 in the pre-survey.
The difference between pre and post scores is your key NSF metric.*

**Question 2 — One thing you learned** *(Paragraph, required)*
```
What is one thing the data showed you today that surprised you?
```

**Question 3 — How did R feel?** *(Checkboxes, required)*
```
How did working in R feel? (Check all that apply)

☐ Scary
☐ Useful
☐ Confusing
☐ Exciting
☐ Like something I could do again
☐ Not for me
```

**Question 4 — Would you try this again?** *(Multiple choice, required)*
```
Would you want to try data science again after today?

○ Yes — I want to keep going
○ Maybe — I need to think about it
○ No — it is not for me right now
```

**Question 5 — What would you change?** *(Paragraph, optional)*
```
Was there a question you wanted to ask that you could not figure out
how to answer? Or anything you wish we had done differently?
```

**Question 6 — Open door** *(Multiple choice, optional)*
```
Would you like to hear about future OLC MST data science workshops
or opportunities?

○ Yes — please keep me in the loop
○ No thank you
```

*If yes — add a follow-up short answer: "Email address (optional)"*
